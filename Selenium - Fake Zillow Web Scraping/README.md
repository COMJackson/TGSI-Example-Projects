# LinkendIn Job Save Bot
### Purpose: 
The purpose of this script is to scrap the data off of
a clone of Zillow.

### How to run.
1. Install Python and and listed modules if not already done.
3. Insert your form URL into the FORM_URL constant.
2. Open *main.py*
3. Run *main.py*
